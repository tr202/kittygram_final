# Kittygram

Kittygram социальная сеть владельцев домашних котиков.
## Возможности
- Публикация и  редактирование профиля питомца
- Размещение фото и достижений котика.

## Технологии
- React
- Django
- DRF

## Infra
- Nginx
- Gunicorn
- Docker
- Docker-compose

## CI/CD
- Github Actions
